# 12V-DC-to-220V-AC-inverter-using-arduino


An Arduino-based DC to AC inverter using MOSFET switching and SPWM control to generate a 220V AC output from a 12V DC source.


## Features
- Arduino Nano based SPWM generation
- MOSFET H-Bridge switching
- 12V DC to 220V AC conversion
- Embedded switching control
- Step-up transformer integration
- Experimental waveform analysis


## Components Used
- Arduino Nano
- IRF3205 MOSFETs
- 7805 Voltage Regulator
- Step-up Transformer (12V-220V)
- Capacitors and Resistors
- 12V Battery Supply


## Working Principle
The inverter uses SPWM signals generated by an Arduino Nano to control MOSFET switching in an H-Bridge configuration. The switching sequence produces an alternating waveform which is stepped up through a transformer to obtain approximately 220V AC output.

- Small AC loads

- DC to AC conversion systems
